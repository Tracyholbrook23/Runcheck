## Bugs / Risks

- **`notifCooldowns` map will grow unboundedly** — Each unique run creates 3+ keys on `users/{uid}.notifCooldowns`. Power users will accumulate a large map over time, potentially hitting Firestore's 1 MB doc limit. Migration plan documented in BACKEND_MEMORY.md Known Issue #9. Not urgent for launch; address before ~500 active users.
